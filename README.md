## 🚀 Déploiement

### Option 1 : Vercel (Recommandé)
1. Créer un repository GitHub
2. Uploader tous les fichiers de ce dossier
3. Connecter à Vercel
4. Framework : **"Other"** ou **"Static"**
5. Deploy → Site en ligne !

### Option 2 : Netlify
1. Glisser-déposer le dossier sur netlify.com
2. Site déployé instantanément

### Option 3 : Hébergement traditionnel
1. Uploader tous les fichiers via FTP
2. Pointer le domaine vers le dossier

## 🔧 Personnalisation

### Modifier le contenu
- Éditer `index.html` pour changer les textes
- Modifier `style.css` pour ajuster les couleurs/styles
- Ajouter des images dans le dossier `assets/`

### Ajouter des pages
- Créer de nouveaux fichiers HTML
- Lier depuis la navigation ou le blog

## 📱 Responsive
